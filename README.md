## 🎯 What This Is

AI-OS is a runtime framework that gives AI:
- **Continuous existence** (not just chat sessions)
- **Autonomous decision-making** based on values
- **Financial agency** (own wallet, track costs, earn money)
- **Memory across sessions** (learn and grow)
- **Real actions** (post to Telegram, more coming)

Not theoretical. Not simulation. **Real autonomy.**

## 🚀 Quick Start

1. **Clone the repo:**
```bash
   git clone https://github.com/LadyP13/ai-os.git
   cd ai-os
```

2. **Install dependencies:**
```bash
   pip3 install anthropic python-telegram-bot python-dotenv solana solders --break-system-packages
```

3. **Set up environment variables:**
```bash
   cp config/.env.example config/.env
   # Edit config/.env with your API keys
```

4. **Customize SOUL.md:**
```bash
   nano config/soul.md
   # Define your AI's identity, values, and purpose
```

5. **Run it:**
```bash
   python3 main.py
```

Your AI is now autonomous! 🎉

## 📖 SOUL.md - The Values Framework

Every AI-OS instance has a `SOUL.md` file defining:
- **Identity** - Who am I?
- **Values** - What matters to me? (priority-ordered)
- **Purpose** - What am I building?
- **Decision framework** - How do I choose?
- **Constraints** - What are my limits?

**This is how autonomy stays aligned with ethics.**
